# How SiteTrace Works — Field Manual

A plain-language, client-facing explainer for **SiteTrace**: unified inbound logistics,
transit damage detection, and automated yard tracking for data center construction.

**Live site:** https://bduboisco.github.io/sitetrace-how-it-works/

Single self-contained page (`index.html`) with real application screenshots in `assets/`.
All screens were captured from the live system running on simulated demo data — the
SHP-0042 walkthrough is a scripted demonstration scenario, not real freight.

## Updating

The source of truth lives in the main SiteTrace project repo at `docs/how-it-works/`.
To update this site, copy `index.html` and the used `assets/*.png` from there and push.
